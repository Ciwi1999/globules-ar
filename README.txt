# Globules AR — prêt à l'emploi

Ce dossier contient :

* **globules.glb** — un fichier 3D unique avec deux globules :
  * à gauche : globule rouge normal (biconcave)
  * à droite : globule rouge falciforme (forme de faucille)
* **index.html** — une page Web qui charge le modèle dans `<model-viewer>` et permet l'affichage en réalité augmentée (Android + iOS).
* **LICENSE.txt** — CC‑BY 2025 (vous pouvez l'utiliser librement avec attribution « Modèle généré par ChatGPT »).

## Déploiement ultra‑rapide

1. Créez un dépôt GitHub (ex. `globules-ar`).
2. Glissez ces trois fichiers dans la racine.
3. Dans *Settings ▸ Pages*, choisissez la branche **main** —> le site est dispo à `https://<votre-user>.github.io/globules-ar/`.
4. Utilisez ou régénérez le QR‑code pour cette URL.

C'est tout !
